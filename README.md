# Proyecto-Final

#Idea Principal

Crear un bot de Discord que promueva la conciencia ambiental y hábitos sostenibles a través de herramientas interactivas, como calcular la huella de carbono según los hábitos diarios, ofrecer alertas y noticias relevantes, proponer desafíos semanales, y generar reflexiones o recomendaciones sobre acciones cotidianas (como reciclar), con el objetivo de educar y motivar a los usuarios a cuidar el medio ambiente de manera divertida y práctica.
